تقرير شامل لكل الـ APIs — تطبيق الراكب (VeeGo)
________________________________________
📌 ملاحظة مهمة — Base URL
كل الـ requests لازم تبدأ بـ /api/ قبل أي endpoint:
Base: https://your-domain.com/api
________________________________________
🔐 1. المصادقة (Auth)
Method	Endpoint	الغرض	Request Body
POST	/auth/register	تسجيل راكب جديد	{ name, email, phone, password }
POST	/auth/login	تسجيل دخول	{ email, password }
POST	/auth/send-otp	إرسال OTP	{ phone }
POST	/auth/verify-otp	التحقق من OTP	{ phone, code }
POST	/auth/refresh	تجديد الـ token	{ refreshToken }
POST	/auth/forgot-password	نسيت الباسورد	{ email }
POST	/auth/reset-password	إعادة تعيين الباسورد	{ token, newPassword }
Response عند Login:
{ "accessToken": "...", "refreshToken": "..." }
⚠️ احفظ الـ token باسم accessToken في AsyncStorage — ده الاسم الصح المستخدم في كل الـ requests
________________________________________
👤 2. البروفايل (Profile)
Method	Endpoint	الغرض	Request Body / Notes
GET	/users/me	جلب بيانات البروفايل	— يرجع { name, email, phone, dob, avatar }
PATCH	/users/me	تعديل الإيميل وتاريخ الميلاد	{ email?, dob? } — name وphone للقراءة فقط
PATCH	/users/me/password	تغيير الباسورد	{ currentPassword, newPassword }
POST	/users/me/avatar	رفع صورة البروفايل	multipart/form-data — field: avatar — يرجع { avatarUrl }
GET	/users/me/stats	إحصائيات	يرجع { savedAmount: number }
GET	/users/me/rating	التقييم	يرجع { overallScore, totalRatings, distribution: { 5:%, 4:%, ... } }
GET	/users/me/notifications	إعدادات الإشعارات	يرجع { notifTrips, notifPromos, notifSystem, notifDriverUpdates }
PATCH	/users/me/notifications	تحديث إعداد إشعار	{ notifTrips?: bool, notifPromos?: bool, ... }
DELETE	/users/me	حذف الحساب	—

🎫 5. الدعم الفني (Support Tickets)
⚠️ الـ Schema مختلف — يجب التصحيح
Endpoint: POST /support/tickets
❌ ما بيبعته تطبيق الراكب:
{ "issueType": "issue_payment", "message": "..." }
✅ ما يتوقعه الـ backend:
{ "subject": "...", "message": "...", "category": "payment" }
جدول تحويل issueType → subject + category:
issueType (تطبيق الراكب)	subject	category
issue_booking	"مشكلة في الحجز"	"other"
issue_payment	"مشكلة في الدفع"	"payment"
issue_driver	"مشكلة مع السائق"	"quality"
issue_app	"مشكلة تقنية"	"other"
issue_other	"استفسار عام"	"other"
الكود المقترح:
const issueMap = {
  issue_booking: { subject: "مشكلة في الحجز",   category: "other"   },
  issue_payment: { subject: "مشكلة في الدفع",   category: "payment" },
  issue_driver:  { subject: "مشكلة مع السائق",  category: "quality" },
  issue_app:     { subject: "مشكلة تقنية",      category: "other"   },
  issue_other:   { subject: "استفسار عام",       category: "other"   },
};
const mapped = issueMap[issueType] ?? issueMap.issue_other;
await axios.post("/api/support/tickets", {
  subject:  mapped.subject,
  message:  userMessage,
  category: mapped.category,
});
Response:
{ "id": 12, "subject": "...", "message": "...", "status": "open" }

🚪 8. تسجيل الخروج (Logout)
لا يوجد endpoint خاص — يتم محلياً:
await AsyncStorage.clear();
// أو
await AsyncStorage.removeItem("accessToken");
await AsyncStorage.removeItem("refreshToken");
